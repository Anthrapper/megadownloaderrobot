## MegaDownloaderRobot

Bot De telegram para descargar archivos de Mega y subirlo a los servidores de Telegram


## Steps to deploy 
```clone project
composer install
chmod 777 -R storage/ bootstrap/cache
set .env
php artisan key:generate
php artisan migrate
sudo apt install -y megatools
```
